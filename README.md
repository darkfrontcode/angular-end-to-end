Angular End-To-End
=====================

## how to install

```
npm install
npm run production
open http://localhost:3000
```

* angular
* angular-ui-router

* babel-preset-stage-0
* express
* jade
* protractor
* webpack
* webpack-dev-middleware
* webpack-hot-middleware
